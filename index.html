Projects

Data Pipeline (Python + SQL)

Implemented an automated data transfer system at Synchem Pharmaceuticals Pvt. Ltd. using Python and SQL ELT methodology, integrated with Task Manager for scheduled executions, and included comprehensive error handling mechanisms for data integrity and reliability.

Pharmaceutical Secondary Sales Analysis

The project streamlined direct sales processes, employing analytical methods and automated data transfer for real-time insights into sales patterns and market dynamics at both state and district levels.

Telegram Bot with Python

I developed a Telegram bot for Synchem that allows users to access personalized documents and their own sales data using commands. The bot is built with a Python backend and is hosted on PythonAnywhere. It operates 24/7 to ensure continuous availability.

Excel Power Pivot Data Modeling

In today's data-driven world, efficiently managing and analyzing large datasets is crucial, especially in the pharmaceutical industry. Leveraging tools like Excel, SQL Server and Power Pivot, our project focuses on optimizing data handling and reporting through effective data modeling techniques.

Work Management on SQL Server Reporting Services

Developed a centralized reporting system using SQL Server Reporting Services (SSRS) to replace Excel-based processes, enhancing real-time data accessibility and personalized user access. This project significantly improved operational efficiency and facilitated data-driven decision-making within the company.

Work Management: Jira Management Portal

The Jira Management web portal for the "Lalit" organizes tasks into Weekly, Monthly, Quarterly, and Yearly columns, detailing activities such as sales data preparation, planning, reconciliation, and updates. Tasks are tracked with specific IDs, priorities, and activity logs, with none marked as done currently.

Interactive District Maps with Folium and GeoPandas

Creating interactive maps is a powerful way to visualize geographic data. With tools like Folium, GeoPandas, and Matplotlib, you can create dynamic and informative maps that display data such as sales and population across different districts.
