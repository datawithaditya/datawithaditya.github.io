Telegram BOT with Python

Published on: June 14, 2024

As part of my recent work, I developed a comprehensive Telegram bot using Python, aimed at centralizing information and communication channels within an organization. This bot simplifies access to relevant data and resources, enhancing user experience and operational efficiency.

Key Features

The Telegram bot offers the following functionalities:

  • Start: Initiates the bot and provides a welcome message.
  • Career: Provides information about current job openings and includes a link to a Google Form for applicants interested in joining the company.
  • My Info: Allows users to request their personal information stored on the company server.
  • Primary: Similar to "My Info," this option enables users to access Daily Primary Sales data specific to their roles.
  • Get Document: Facilitates the retrieval of specific documents of current Quarter Scheme from the company server.

Note: The "My Info," "Primary," and "Get Document" options enable users to request and receive their files directly via Telegram. The document sender system handles these requests.

Document Sender System Workflow

    1. A user requests a document through the Telegram app.
    2. The backend Python code retrieves the user's Telegram ID.
    3. The system compares this Telegram ID with existing user IDs in the company server.
    4. Upon matching the ID, the system identifies the employee's name associated with the ID.
    5. The requested document is then pushed to the user's Telegram app via the backend.

Technical Implementation

  • Backend Development Tech:Python
  • Libraries:
      1. python-telegram-bot: For Telegram bot functionalities.
      2. Pandas:For Document Sender System
