Pharmaceutical Secondary Sales Analysis

Published on: May 21, 2024

Project Overview

This project was designed to enhance direct sales processes between Stockists/Distributors and Retailers. By employing advanced analytical methods and automated data transfer mechanisms, the project aimed to provide near real-time insights into sales patterns and market dynamics at both state and district levels.

Key Features

    1. Analytical Methods
      • Graphical and Tabular Data Analysis: The project utilized various analytical methods, including graphical and tabular representations, to identify sales patterns.
      • Granular Analysis: Filters by product group and employee hierarchy allowed for detailed analysis, with further segmentation into individual product SKUs and employee roles.
    2. Automated Data Transfer
      • Near Real-Time Access: Leveraging automated data transfer mechanisms ensured near real-time access to updated sales data. This facilitated agile decision-making and strategy development.
    3. State and District Analysis
      • State Maps: The project examined the companyâ€™s active states, further dissected into districts. Options included viewing sales data for the current financial year and population data according to the 2011 Census.
      • Active States: The project covered five states: Madhya Pradesh, Uttar Pradesh, Maharashtra, Rajasthan, Chhattisgarh, and Uttarakhand.
      • Navigation: Users could navigate to any of these states via the dashboard.

Dashboard Elements

    1. Navigation
      • State Maps Navigation: This feature allowed users to navigate to different state maps, providing an intuitive way to explore data.
    2. Product Group Comparison
      • Sales Comparison: The dashboard facilitated unit-wise sales comparisons with previous financial year sales, allowing for a quick assessment of performance.
    3. Sales Visualization
      • State Sales Map: A map visualization showing sales distribution across states provided a clear overview of regional performance.
      • Line Chart with Historical Data: This chart featured historical sales data, with MSD (Manager-Sales & Development) as the legend, helping track performance trends over time.
    4. Filtering Options
      • Advanced Filters: Users could filter dashboard elements by employee name, employee hierarchy, product group, short name of the product, and headquarter name. This functionality enabled tailored data views to meet specific analytical needs.

Conclusion

By integrating sophisticated analytical tools and near real-time data access, this project significantly streamlined direct sales processes. It empowered stakeholders with detailed, actionable insights, driving better market strategies and improved sales performance.
